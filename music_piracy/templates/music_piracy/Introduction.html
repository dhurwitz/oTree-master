Welcome to the oTree demo game!

This is a simple 1-player game demonstrating some of oTree’s basic capabilities, as well as its interaction with some plugins. This game does not have any experimenter manual input, so you just need this tab. Experimenter input is an oTree feature that allows an experimenter to give input (such as say the draw from a physical Ellsberg urn).
